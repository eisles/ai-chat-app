# Product Overview

AI チャットと画像検索を中心に、AI モデルとデータベース連携を検証できるデモアプリ。開発者が UI と API を通じて AI 連携の体験を試せることを目的とする。

## Core Capabilities

- AI Gateway 経由のチャット体験提供
- 画像から説明文を生成し、類似テキスト検索を行う API
- 複数プロバイダ（OpenAI / Gemini / ローカル）の画像埋め込みと保存
- 画像埋め込みの検索・一覧 UI（OpenAI / Vectorize 等の比較）
- 検索対象テキストの登録・一括登録
- UI から API の動作確認ができる簡易操作画面

## Target Use Cases

- AI モデル連携の UI/UX を確認したい
- 画像説明生成＋ベクトル検索の PoC を素早く試したい
- 画像埋め込みモデルの比較検証を進めたい
- Neon + pgvector の利用フローを検証したい

## Value Proposition

チャット・画像検索・登録 API が同居しており、AI と DB の統合フローを 1 つの Next.js アプリ内で素早く検証できる。

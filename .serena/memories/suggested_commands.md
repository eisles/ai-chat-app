# Suggested Commands
- `npm run dev`: 開発サーバー起動
- `npm run build`: 本番ビルド確認
- `npm run lint`: ESLint 実行
- `npm run test`: Vitest watch
- `npm run test:run`: Vitest 単発実行
- `cargo check`: 承認済み。Rust 関連タスクがあれば使用可能だが、このリポジトリの主要スタックは TypeScript。
- `rg <pattern> app lib tests`: 高速検索
- `find app -maxdepth 2 -type d | sort`: 画面/API 構成の把握
- `git status --short`: 作業差分確認
